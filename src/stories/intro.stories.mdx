## Getting Started

Classmate는 실시간으로 강연자와 청중을 연결해주는 크라우드소싱 플랫폼입니다. 스토리북을 통해 UI에 사용되는 컴포넌트들을 테스트 해볼 수 있습니다.

### Install

```
git clone https://github.com/Dev-TeamOne/classmate-frontend.git
yarn && yarn storybook
```
